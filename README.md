# Koa file uploading ![](https://api.travis-ci.org/b17/koa-file-uploading.svg?branch=master)

Generates a secure (random) link to the uploaded file.

## Requirements

- Node.js v10+

## Getting started

- `npm i`
- `node index.js`
- Open your browser at http://localhost:3000
